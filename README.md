# ODArchiver

* Client:Server Architecture to crawl web ressources

## prerequisits

* npm and node
* a running mongodb

## Quick Start

Get started developing...

```shell
# install deps
npm install

# run in development mode
npm run dev

# run tests
npm run test
```

---

### Run in *production* mode

* provide .env file

Compiles the application and starts it in production mode.

```shell
npm run build
npm start
```
